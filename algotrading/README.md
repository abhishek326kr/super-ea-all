# AlgoTradingBot.online

This is a Next.js 14+ project initialized with Tailwind CSS and TypeScript.

## Getting Started

First, install dependencies (if you haven't already):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `components/Header.tsx`: Sticky navigation header with scroll effects and mobile menu.
- `components/Footer.tsx`: 4-column footer with social links and responsive layout.
- `app/layout.tsx`: Main layout including global font (Outfit) and Header/Footer.
- `app/page.tsx`: Demo homepage with Hero section.
- `app/globals.css`: Global styles, themes, and animations.

## Features

- **Header**: Sticky, transparent to dark/blurred on scroll, mobile drawer, neon green accents.
- **Footer**: Premium dark style, clean typography, SVG social icons.
- **Styling**: Tailwind CSS v4 with custom neon theme colors.
- **Font**: Outfit (Google Font) integration.
